# UI Reference Explorations

- Status: reference
- Source of truth: `docs/ui/operator-console.md`, `docs/ui/visual-language.md`

## Purpose

This directory stores exported UI explorations, generated HTML, screenshots, and generator briefs.

These files are useful for:

- implementation inspiration
- visual comparison
- preserving early exploration artifacts

These files are not the canonical source of truth for product behavior.

## Rules

- `generator-design-brief.md` files are preserved exports from the design generator.
- `prototype.html` files are reference prototypes, not production code.
- `preview.png` files are visual snapshots for comparison during implementation.
- When an idea graduates into a real product rule, move that rule into [operator-console.md](/Users/cliff/workspace/Spec2Flow/docs/ui/operator-console.md) or [visual-language.md](/Users/cliff/workspace/Spec2Flow/docs/ui/visual-language.md).
